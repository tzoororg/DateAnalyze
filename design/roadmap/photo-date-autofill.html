Us ♥ — #16 · auto-fill “When” from a photo Tiny

Adding a photo to the log/edit form reads its EXIF capture date (js/exif.js already does this for bulk import) and sets the “When” date to match — only while the date is still the untouched default, so it never clobbers a date you picked by hand. A small chip explains what happened and lets you undo.

Where it plugs in. onPhotoPick() in js/ui.js already handles photo adds; after storing the blob, call readExif(file) (from js/exif.js) on the first added photo. If it returns a date and the form date is still the default (fresh entry, user hasn’t edited it), set draft.date + the #f-date input and show the chip. No schema change, no new dependency — the EXIF reader is already shipped.
